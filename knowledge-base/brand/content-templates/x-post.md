# X (Twitter) Post Template

## Format Guidelines
- Length: Under 280 characters for single posts. Use threads for longer ideas (3-7 tweets max).
- Tone: Sharp, direct, opinionated. No corporate fluff.
- First tweet must stand alone — it needs to work even if nobody reads the thread.
- Use concrete numbers over vague claims.
- No hashtags in the main text (they look spammy on X). One or two at most, only if relevant.

## Single Post Template

```
[BOLD CLAIM or OBSERVATION — under 280 characters]
```

## Thread Template

```
Tweet 1: [HOOK — provocative claim or surprising stat that stands alone]

Tweet 2: [CONTEXT — why this matters right now]

Tweet 3-5: [INSIGHT — the breakdown, one point per tweet. Use → or numbered lists]

Tweet 6: [TAKEAWAY — so what? What should the reader do with this?]

Tweet 7: [CTA — follow for more, check the link, reply with your take]
```

## Example Single Post

Your marketing stack has 7 tools and zero visibility.

That's not a tech stack. That's a Rube Goldberg machine.

## Example Thread

**Tweet 1:**
Hot take: Most mid-market marketing teams would get better results with fewer tools, not more.

**Tweet 2:**
Here's why — every tool you add creates a gap. Data doesn't flow. Context gets lost. Your SDR has no idea why a lead is "hot."

**Tweet 3:**
What actually moves pipeline:
→ One workflow engine (not 5 Zapier chains)
→ Intent signals connected to routing (not sitting in a dashboard)
→ Attribution that goes from first touch to revenue (not just MQLs)

**Tweet 4:**
The best GTM teams we see aren't the ones with the biggest budgets. They're the ones with the fewest handoffs.

**Tweet 5:**
Less tools. Less gaps. More pipeline. That's it.

## Image Prompt Guidelines for X
- X images should be bold and simple — one visual idea, not a collage
- Prefer abstract/conceptual over literal illustrations
- High contrast works well in the feed
- Include dimensions in the prompt: 1600x900px (landscape) or 1080x1080px (square)

### Example Image Prompt
```
---
**Image Prompt:**
Abstract minimal illustration: five disconnected circles in muted gray, converging into one bright teal circle through flowing lines. Dark background, clean vector style. No text.
**Dimensions:** 1600x900px
```
